# 🔐 Multisig ASA Management on Algorand 

This project demonstrates how to securely manage Algorand Standard Assets (ASA) using a **2-of-3 multisignature account** on the Algorand blockchain.

## 📌 Features

- Sets up 3 random accounts and funds them via localnet dispenser
- Creates a **2-of-3 multisig account** from the 3 accounts
- Funds the multisig account
- Creates an ASA (Algorand Standard Asset) with multisig as manager, reserve, freeze, and clawback
- Updates the **freeze address** of the ASA using a multisig-configured transaction
